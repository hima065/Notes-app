Notes App

A full-stack Notes Management Application built using React.js, Node.js, Express.js, and MySQL. The application allows users to create, view, update, and delete notes through a simple and user-friendly interface.

Features

- Add Notes
- View Notes
- Update Notes
- Delete Notes
- MySQL Database Integration

Tech Stack

- React.js
- Node.js
- Express.js
- MySQL
- Axios

Run Project

Backend

cd backend
npm install
node server.js

Frontend

cd frontend
npm install
npm run dev
